# Arduino-Maze-Solver
Arduino Car that solves simple mazes via the hand-on-wall-rule

Modules used:

Arduino Uno x1

L298n motor driver x1

Battery x2

breadboard x1

HC-SR04 ultrasonic sensor x3

Example of maze Solver doing its thing: https://youtu.be/MwYncLcmyl8
